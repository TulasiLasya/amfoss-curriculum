# Melofi 

Melofi is a basic mobile music player application developed as part of the amFOSS Curriculum Task 07.

The aim of this project is to understand the fundamentals of Android app development including user authentication and media playback.

## Features

- User Registration
- User Login
- Basic Music Playback
- Simple User Interface for playing songs

## Current Status

The project is still under development.

Currently, the authentication system and basic music player functionality have been implemented successfully in the Android Emulator.

## Known Issue

The music player works properly in the Android Emulator but currently does not work on a physical mobile device. This issue is being investigated and will be resolved in future improvements.

## Technologies Used

- Android Studio
- Kotlin
- Android SDK

## Learning Outcomes

Through this project, I learned:

- Basics of Android app structure
- Implementing user authentication
- Handling media playback in Android
- Debugging issues between emulator and real devices
