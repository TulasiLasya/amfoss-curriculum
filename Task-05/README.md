## Frontend Development

This task was done with React.js a Javascript library, which is useful for building User Interfaces by component wise.

I have made the user interface to MeloFi music website according to the designs that i had made in task-04.
You can find my Frontend of this MeloFi application in React_app folder.

#### Things learnt by doing this Task:

* Got idea on creating components using react by defining functions and exporting them which includes basic javascript and html.
* Designing the webpages using css which mainly includes concepts like: flex-box model, gradiation, classes, font, ..etc.
* About useState, useEffect and rendering concepts.
* About React icons 
* Routing the webpages finally to make the application interactive.
